# Changelog

Все значимые изменения в проекте будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.0.0/).

## [1.1.0] - 2026-03-27

### Добавлено
- Gunicorn как production WSGI сервер
- Функция очистки JSON от комментариев `/* ... */` для совместимости со старыми версиями TRASSIR

### Изменено
- Dockerfile теперь использует gunicorn вместо flask dev server
- Запуск фонового потока сборщика метрик вынесен на уровень модуля

### Исправлено
- Предупреждение "WARNING: This is a development server"
- Ошибка парсинга JSON с комментариями на сервере KZN

## [1.0.0] - 2026-03-25

### Добавлено
- Первый релиз
- Базовый экспортер метрик TRASSIR
- Поддержка 9 TRASSIR серверов
- Метрики: статус сервера, камеры, CPU, архивы
- Docker контейнер
- GitHub Actions CI
